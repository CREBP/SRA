FAQ

1
What is Batt?
Batt is a tool to define complex data collection forms. You can use Batt to create complex, hierarchical input systems.
2
Who makes Batt?
Batt is created and maintained by MFDC as a platform to make programming long and painful forms easier.
3
What is the license on Batt?
Batt uses the Creative Commons Attribution Non-Commercial No-Derivative v3 license.
This means you can use Batt for non-commercial projects as much as you want. If you wish to use it commercially though you will need to contact us.

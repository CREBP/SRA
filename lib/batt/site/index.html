Batt is a modular form plugin library for creating complex, hierarchical validated forms.

Features

HTML5 responsive design

Batt is compatible with many different frameworks including Bootstrap, jQuery Mobile and Sencha and has plugins for most of the popular JavaScript plugins.

More info

Extremely simple to learn

In its simplest form Batt can be thought of as a simple markup language for creating powerful forms. It has built in support for extremely complex inputs such as input validation and media uploading which means it can be extended easily.

More info

Modular design

Batt has an extremely modular design. You can pull in just the plugins you require to style your forms exactly how you want them.
